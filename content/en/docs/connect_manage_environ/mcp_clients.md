Within topology, MCP Clients are used to manage MCP Client Types that will be used with MCP Servers.

## MCP Clients management

### Viewing available MCP Client Types

1. Navigate to *Topology > MCP Clients*.

The list of MCP Client Types is displayed with the following information:

* Client name - The title of the client type.
* Referenced - The number of times the client type is referenced.
* Updated - Date of the last modification.
* Action menu - The ellipsis menu to manage the clien type
    * Edit - Edit the client type
    * Delete - Delete the client type

### Creating a MCP Client Type
1. Navigate to *Topology > MCP Clients*.
2. Click **+ Add Client Type** to add a new client type.
3. Add the following and then click **Create**:

    * Client Type - The display name for the MCP Client Type in the WebUI
    * Image - An icon or image to be associated with the MCP Client Type

## Edit a MCP Client Type
1. Navigate to *Topology > MCP Clients*.
2. From the list view, find the Client Type to edit.
3. Select **Edit** from the action menu

## Delete a MCP Client Type
1. Navigate to *Topology > MCP Clients*.
2. From the list view, find the Client Type to delete.
3. Select **Delete** from the action menu
