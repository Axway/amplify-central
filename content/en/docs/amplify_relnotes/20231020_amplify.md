### Asset Catalog

The following enhancements and bug fixes are available in this release.

#### Feature updates for Asset Catalog

None

#### Bug fixes for Asset Catalog

* **Access rights tab always selects restricted** - Depending of the user role who is creating/updating a product, the *Access Rights* step automatically selects either the Restricted area (for Central Admin users) or the Shared area (for user having a team role). USer who are not Central Admin cannot select the Restricted option.

### Product Foundry

The following enhancements and bug fixes are available in this release.

#### Feature updates for Product Foundry

* **Migrate Subscription from archived plan** - From Central > Product Foundry > Plan list, it is now possible to access the *Migrate* menu when a plan is already archived to be able to migrate remaining subscription. As soon as there are no more active subscription, the *Migrate* menu is disabled.

#### Bug fixes for Product Foundry

* **Category Visibility column width issues** - The truncation of this column has been fixed to see the entire value.
* **Access rights tab always selects restricted** - Depending of the user role who is creating/updating a product, the *Access Rights* step automatically selects either the Restricted area (for Central Admin users) or the Shared area (for user having a team role). USer who are not Central Admin cannot select the Restricted option.
* **Document library allows to create Markdown document** = The Document Library is now capable of creating Markdown document. Those document can be re-use in the Product Documentation or in Marketplace settings (Footer / Menu link) as other regular Document Library documents.

#### Known issues for Product Foundry

There are no known limitations in this release.

## Consumer experience

### Marketplace

The following enhancements and bug fixes are available in this release.

#### Feature updates for Marketplace

* **Access request Subscription selection** - Since it is no more possible to request multiple access to a resource via the same subscription, the subscription dropdown selection has been enhance to allow select only subscription that does not already have an access to the resource. For subscription already having access to the resources, they are simply disabled from the list and a tooltip indicates the number of application already having access. The tooltip also contains the link to the concerned application list.

#### Bug fixes for Marketplace

* **Deleted plan still visible in Marketplace** - It is possible that some synchronization between Marketplace and product Foundry failed to be processed which lead to orphan plan in Marketplace. Now those orphan plan are no more visible in the Marketplace.
* **Homepage hero content alignment is wrong** - with the change on the Marketplace theming we also updated the home page hero content alignment (left / center / right) to respect what the user sets.

#### Known issues for Marketplace

There are no known limitations in this release.

### Consumer insights

The following enhancements and bug fixes are available in this release.

#### Feature updates for Consumer insights

### Unified Catalog announcement

The statistics related to Unified Catalog have been turned off in preparation of the deprecation of the Unified Catalog feature. Users can no longer see how many times a Unified Catalog item has been viewed (from the list and the catalog item details views).

A migration script is available on [Axway GitHub](https://github.com/Axway/unified-catalog-migration).
