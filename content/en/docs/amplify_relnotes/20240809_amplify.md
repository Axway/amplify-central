### Axway Agent SDK

The following enhancements and bug fixes are available in this release.

#### Enhancements for Agent SDK

* **Marketplace only support** - Agent SDK version 1.1.94 or later only has support for the Marketplace.  Prior Agent SDK versions no longer support the Unified Catalog.
* **Improved transaction processing** - Agent SDK has been improved to include a default mretric event reporting interval of hourly.  The ability to sample all errors has been removed to reduce the amount of transaction processing in situations with a high error transaction rate.  This has been replaced with a sampling rate which has a default of zero percent to allow for the sampling rate to be applied to errors now. 

#### Bug fixes for Agent SDK

There are no bug fixes in this release.
