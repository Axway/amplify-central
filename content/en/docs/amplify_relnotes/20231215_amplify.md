### Product Foundry

#### Feature updates for Product Foundry

* **Limit the product plan time** - new option is available when creating a paid plan to limit the number or renewal (unlimited by default). Like this is possible for the consumer to subscribe for a limited period of time and be sure at the end of the contract that the subscription will be terminated.

* **CSV files exported from Amplify Business Insights API Health screen does not include all data** - API limits for the CSV export has been increased to 1000 which will allow the CSV file to represent all data.

## Consumer experience

### Marketplace

The following enhancements and bug fixes are available in this release.

#### Feature updates for Marketplace

* **Limit the product plan time** - When viewing a plan, the number of term for which the plan can be used is display in the plan details view. The subscription details screen has been reorganized to show when the subscription will/has expired.
* **Remove corrupted resources from Active plan** - when an API Service is removed but was used in a product plan, the product plan becomes corrupted. It is now possible when viewing the plan quota to remove the corrupted resources using the in icon. The bin icon is also present when the plan is in draft state.

#### Bug fixes for Marketplace

* **HTTP Basic not available when authorizing** - With the introduction of the new *Authentication* side panel, it is now possible, for any type of credential, to test the API by providing values that have not been provision from Marketplace.

#### Known issues for Marketplace

There are no known limitations in this release.

### Consumer insights

There are no new enhancements or bug fixes in this release.

### Unified Catalog announcements

* The statistics related to Unified Catalog have been turned off in preparation of the deprecation of the Unified Catalog feature. Users can no longer see how many times a Unified Catalog item has been viewed (from the list and the catalog item details views).
* A migration script to move to Marketplace is available on [Axway GitHub](https://github.com/Axway/unified-catalog-migration).
