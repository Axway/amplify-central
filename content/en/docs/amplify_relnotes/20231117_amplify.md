## Consumer experience

### Marketplace

The following enhancements and bug fixes are available in this release.

#### Feature updates for Marketplace

* **Oauth authentication flow support** - It is now possible to request an OAuth token from the Marketplace Try out screen. This will navigate the user to the authentication screen and back to the Marketplace once the authentication succeed.
* **ISO date format** - All Marketplace screens displaying a date format (Product details / Subscription / Credentials) are now using the YYY-MM-DD ISO format.

#### Bug fixes for Marketplace

* **Access request list not displayed correctly** - When using arrays inside a field, the items are now seen as multiple line instead of a single line coma separated.
* **Search filed from Marketplace home page does not return any result** - the url referencing the search has been corrected to properly take the user entry as an url parameter.
* **Category search does not return associated product** - In the Marketplace Category listing, selecting a category now returns the assigned products.
* **Missing showing the number of item in application page** - The number of existing applications is now displayed at the top and the bottom of the page.

#### Known issues for Marketplace

* **Issue** - Description.

### Unified Catalog announcement

The statistics related to Unified Catalog have been turned off in preparation of the deprecation of the Unified Catalog feature. Users can no longer see how many times a Unified Catalog item has been viewed (from the list and the catalog item details views).

A migration script to move to Marketplace is available on [Axway GitHub](https://github.com/Axway/unified-catalog-migration).
