### Topology

The following enhancements and bug fixes are available in this release.

#### Feature updates for Topology

* **Edit SaaS AWS Service configuration** - SaaS AWS Service configuration parameters can now be edited from the WebUI.  This includes all the parameters which were entered during the creation of an AWS SaaS environment.

#### Bug fixes for Topology

* **Compliance Profiles Pagination** - A fix was made to the Compliance Profiles page to display the correct page when the user clicks on another page from the WebUI.
* **Environment and Stage wizard error message** - A fix was made to improve the error message when removing an assigned stage on the WebUI.  The error message now displays "At least 1 APIServiceInstance references a stage removed from this Environment".

#### Known issues for Topology

The are no known issues in this release.

### Asset Catalog

The following enhancements and bug fixes are available in this release.

#### Feature updates for Asset Catalog

There are no new enhancements in this release.

#### Bug fixes for Asset Catalog

* **Asset Created without any API resources** - A fix was made to prevent "Save and Activate" action from the WebUI of an asset with no grouped API resources.
* **Asset Deletion error** - A fix was made when to enable the Asset deletion action from the WebUI.  The error message displayed was "AssetRequestDefinition was not found" and the asset was not deleted.
* **Grouped Asset Resources** - A fix was made to display all the grouped Asset Resources on the WebUI when editing an Asset.
* **Multiple role access** - A fix was made to allow a user with both the Catalog Manager and Developer role to edit Asset ownership and permissions.

#### Known issues for Asset Catalog

There are no known limitations in this release.

## Consumer experience

### Marketplace

#### Feature updates for Marketplace

* **Product review** - Consumers can now review their satisfaction with a product in the Marketplace. Marketplace Managers must turn on the Product Ratings&Reviews feature to enable Consumers to review a product. Consumer can leave one review per product. Consumer is able to change his review any time. More information [Marketplace ratings and reviews](/docs/manage_marketplace/consumer_experience/ratings_reviews).
* **Consumer sees Active status for subscription until cancelled** - The 2 statuses `Migration Scheduled` and `Archival Scheduled` have been replaced with `Active`. Consumer is still able to see when the subscription will ends thanks to the Expires date field.

#### Bug fixes for Marketplace

There are no bug fixed for Marketplace.

#### Known issues for Marketplace

There are no known limitations in this release.

### Consumer insights

There are no new features or bug fixes in this release.

### Unified Catalog announcements

The statistics related to Unified Catalog have been turned off in preparation of the deprecation of the Unified Catalog feature. Users can no longer see how many times a Unified Catalog item has been viewed (from the list and the catalog item details views).

A migration script to move to Marketplace is available on [Axway GitHub](https://github.com/Axway/unified-catalog-migration).
