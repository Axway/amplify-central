AMPLIFY API Management

{{< /blocks/section >}} {{< blocks/section color="dark" >}} {{% blocks/feature icon="fa-lightbulb" title="API Gateway" url="/docs/api_mgmt_overview/api_mgmt_components/apigateway/" %}} API Gateway manages, delivers, and secures enterprise APIs, applications, and consumers. {{% /blocks/feature %}} {{% blocks/feature icon="fa-lightbulb" title="API Manager" url="/docs/apim_administration/apimgr_admin/api_mgmt_workflow/" %}} API Manager is a web-based API administration and management tool layered on API Gateway. {{% /blocks/feature %}} {{% blocks/feature icon="fa-lightbulb" title="API Portal" url="/docs/apim_administration/apiportal_admin/apip_overview/" %}} API Portal is a self-service developer portal layered on both API Manager and API Gateway. {{% /blocks/feature %}} {{< /blocks/section >}} {{< blocks/section color="primary" >}}

AMPLIFY Central

{{< /blocks/section >}} {{< blocks/section color="secondary">}} {{% blocks/feature icon="fa-lightbulb" title="API Management" url="/docs/central/quickstart" %}} Design a custom API and import it into AMPLIFY Central. Add security to your API and share it with your consumers using AMPLIFY Catalog. {{% /blocks/feature %}} {{% blocks/feature icon="fa-lightbulb" title="DevOps" url="/docs/central/cli_getstarted/" %}} Integrate AMPLIFY Central in your existing DevOps infrastructure using the AMPLIFY CLI or AMPLIFY Central DevOps API. {{% /blocks/feature %}} {{% blocks/feature icon="fa-lightbulb" title="Microservice Mesh Governance (BETA)" url="/docs/central/mesh_management" %}} Centrally manage APIs and microservices across multiple cloud and on-premise environments. {{% /blocks/feature %}} {{< /blocks/section >}} {{< blocks/section color="primary" >}}
